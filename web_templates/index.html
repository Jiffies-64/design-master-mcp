Design Document MCP Server

Server for generating design documents with AI assistance

API Endpoints

Health Check

GET /health

Check if the server is running

Start Document Generation

POST /mcp/start_document

Begin generating a design document

Parameters:

{
    "project_root_path": "/path/to/project",
    "user_id": 123
}

Get Next Step

GET /mcp/next_step?template_id=1

Get the next prompt in the sequence

Submit Placeholder Content

POST /mcp/submit_content

Submit content for placeholders

Parameters:

{
    "placeholder_id": 1,
    "content": "Placeholder content"
}

Generate Complete Document

POST /mcp/generate_document

Generate the complete design document

Parameters:

{
    "template_id": 1
}
